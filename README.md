## Usage
Just download and execute the script :

```sh
wget https://raw.githubusercontent.com/intsez/Pi-hole_autoinstaller/main/Pi-hole_autoinstaller.sh
chmod +x Pi-hole_autoinstaller.sh
./Pi-hole_autoinstaller.sh
```
